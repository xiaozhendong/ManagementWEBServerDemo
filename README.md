### Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:7001/
```

### 部署

```bash
$ npm start
$ npm stop
```

### npm 脚本

- 运行 `npm run lint` 检查代码格式.
- 运行 `npm test` 进行单元测试.
- 运行 `npm run autod` 升级依赖, 详情见 [autod](https://www.npmjs.com/package/autod) .
